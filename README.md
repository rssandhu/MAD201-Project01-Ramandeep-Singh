# Add Transaction Form Example
Smart Budget Tracker Lite is a cross-platform Flutter application designed to help users track their daily income and expenses efficiently. The app offers users an intuitive interface to add and categorize transactions, view historical data, monitor summaries, and customize preferences such as theme and currency.

The app demonstrates key Flutter concepts covered in the course, including multi-screen navigation, state management with setState(), persistent local storage with SQLite and SharedPreferences, and basic API integration to fetch live currency conversion rates.

Key Features
Add transactions with fields for title, amount, type (income/expense), category, and date.

View a chronological list of transactions with color-coded amounts and options to edit or delete.

Monitor overall account health with a dashboard showing total income, expenses, and current balance.

Analyze spending and income by category and by month in a dedicated reports and summary section.

Personalize app experience with theme toggle (light/dark) and currency preference.

Fetch and display live exchange rates for the selected currency using a simple HTTP API.

Technologies Used
Flutter framework for cross-platform UI.

SQLite database (with sqflite and sqflite_common_ffi packages) for local transaction storage.

SharedPreferences for persisting user preferences.

HTTP package for retrieving currency conversion data.

Responsive design supporting mobile, desktop, and web platforms.
